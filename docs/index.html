About This Project
This project is the final assignment for the course GEB2507: AI-Driven Virtual Worlds and Serious Games for Societal Applications at CUHKSZ. Our game, Code Runner, is a puzzle-based educational serious game designed to teach fundamental programming concepts. Players progress through 8 levels by dragging and assembling embedded commands to control four heroes and solve increasingly complex challenges. Throughout the game, players learn essential programming logic, including conditional statements, loops, and algorithmic thinking.
